## Fluxo de Submissao

```
1. Revisar e finalizar todos os documentos acima
2. Converter documentos para PDF (permitir copy/paste)
3. Cadastro na Plataforma Brasil (se ainda nao possui)
4. Credenciamento na Rede Pesquisa EBSERH (GOV.BR)
5. Registrar projeto na Rede Pesquisa EBSERH
6. Preencher formulario de 6 etapas na Plataforma Brasil
7. Upload dos PDFs na etapa 5
8. Gerar, imprimir, assinar e carimbar Folha de Rosto
9. Escanear e fazer upload da Folha de Rosto
10. Submeter ao CEP do HUCAM/UFES
11. Aguardar parecer CEP (~30 dias)
12. Enviar parecer CEP para pesquisaclinica.hub@ebserh.gov.br
13. Aguardar autorizacao final EBSERH
14. Iniciar coleta de dados (SOMENTE apos aprovacao)
```
